# Python-Website-Cloner
A Python tool that downloads a website’s HTML, CSS, JavaScript, and images so the site can be viewed locally.

A simple Python tool that clones websites by downloading HTML, CSS, JavaScript, and image files.

## Features

- Clone webpage HTML
- Download CSS files
- Download JavaScript files
- Download images
- Save website locally
- Easy to use

## Technologies Used

- Python
- Requests
- BeautifulSoup

## Installation

Clone the repository:

git clone https://github.com/yourusername/website-cloner.git

Navigate to project folder:

cd website-cloner

Install dependencies:

pip install -r requirements.txt

## Usage

Run the script:

python cloner.py

Enter the website URL when prompted.

Example:

Enter website URL: https://example.com

The cloned site will be saved in:

cloned_site/
